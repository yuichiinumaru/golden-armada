GOOGLE GEMINI API DOCUMENTATION

VERY IMPORTANT OBSERVATIONS
1 - This is 12/2025, which means your traning is **largely outdated**. You should **not** trust it regarding current most updated LLM models, as this is a subject that progresses very fast.

2 - Google Gemini most recent models are gemini-3-flash-preview and gemini-3-pro-preview. Gemini 2.5 family has flash, flash-lite, and pro variants, and is still stable. **All other models (such as 1.5 and 2.0)are deprecated, and you should never use them**
https://ai.google.dev/gemini-api/docs/gemini-3
https://ai.google.dev/gemini-api/docs/deprecations
(note that 1.5 doesn't even appear anymore in the official documentation as it is now completely gone and no longer supported)

3 - The correct python lib is 'google-genai', not 'google-generativeai' (also deprecated, and never to be used)
https://ai.google.dev/gemini-api/docs/migrate
