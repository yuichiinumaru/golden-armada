# AI & LLM

Found 2 MCP servers in this category.

### gpt-computer-assistant ⚠️
**Description**: Este projeto demonstra como um agente de IA pode interagir com um sistema operacional, simulando um assistente de computador. Ele permite que o agente receba comandos, execute ações no sistema (como navegar em diretórios, criar arquivos ou executar programas) e receba feedback sobre o resultado dessas ações. Essencialmente, transforma o agente em uma interface programável para o computador, abrindo portas para automação de tarefas complexas e interações mais profundas.

**URL**: [https://github.com/Upsonic/gpt-computer-assistant](https://github.com/Upsonic/gpt-computer-assistant)

### model-context-protocol ⚠️
**Description**: Este projeto define um protocolo para comunicação entre agentes de IA e ferramentas externas, permitindo a manutenção de um contexto persistente. Ele estabelece um padrão para que diferentes sistemas possam interagir de maneira estruturada, facilitando a orquestração de tarefas e a troca de informações de forma eficiente. O objetivo é criar um ecossistema onde a IA possa se integrar fluidamente com outras aplicações.

**URL**: [https://github.com/pollinations/model-context-protocol](https://github.com/pollinations/model-context-protocol)
