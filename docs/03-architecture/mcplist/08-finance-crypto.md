### yahoofinance-mcp ⚠️
**Description**: Este projeto permite que agentes de IA acessem dados financeiros históricos e em tempo real do Yahoo Finance. Ele fornece uma interface estruturada para consultar informações como preços de ações, dados de empresas e indicadores econômicos, facilitando a integração de análises financeiras em aplicações de IA. O objetivo é simplificar a obtenção de dados para tomadas de decisão informadas e estratégias de investimento.
