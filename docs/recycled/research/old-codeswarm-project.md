# CodeSwarm: Project Specifications (Legacy)

## Overall Goal
To implement a multi-agent coding system focusing on collaborative code generation, review, and project management using ADK.

## Agent Roles
- **AdminAgent**: Project Manager. Breaks down goals into tasks.
- **DevAgent**: Implementation. Writes code based on tasks.
- **RevisorAgent**: Quality Control. Reviews Dev output.

## Technical Foundation
- **LLM**: Google Gemini.
- **Framework**: ADK (Agent Development Kit).
- **Protocol**: A2A (Agent-to-Agent) communication via JSON.
